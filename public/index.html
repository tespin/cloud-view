Google Cloud View

ABOUT

Google Cloud View is a project by Tristan Espinoza that displays a Google Street View image of the sky above your current location.

Visitors can request a Cloud View image by clicking the 'Submit' button. This prompts the browser for permission to use your current latitude and longitude, sending it to the Google Street View API and returning a Street View image with the camera pointed upwards.
