Google Cloud View

ABOUT

Google Cloud View is a project by Tristan Espinoza that uses the Google Street View API to display an image of the sky above your current location.

Visitors can request a Cloud View image by clicking the 'Request' button. This prompts the visitor for permission to use your current latitude and longitude. The request then returns a Street View image with the camera pointed upwards.
